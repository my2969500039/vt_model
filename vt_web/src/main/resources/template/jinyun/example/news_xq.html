晶云药物邀您莅临CPhI China 2017



CFDA日前对药品审评审批制度的改革对创新药物的研发起到了积极推动的作用，促进市场加大加快创新药物研发的投入及步伐。中国正处在新药研发腾飞的前夜，中国新药研发前途一片光明！但新药研发 仍然是一个“苦活”、“累活”，需要创业的企业家和科学家们一起建立自有核心平台，借助资本的力量及最专业的CRO服务来完成每一个创新小分子药物的开发，推动药物快速进入临床研究。

本次专题研讨会聚集华南地区专注做新药的职业新药掘金者，交流探讨小分子药物开发中的经验教训，项目推进过程中的思考，在突破技术层面障碍的同时关注公司的长期发展与融资渠道。我们期待老朋友 的相聚，新朋友的相识，欢迎大家报名参加！
