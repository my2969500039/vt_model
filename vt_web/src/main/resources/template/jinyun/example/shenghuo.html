学习型组织

中央读书帮 晶云是一个真正的学习型组织，一个有意义有意思的组织，绝世秘籍《结晶腾云四十二章经》助你允文 允武，超越自我：日有所行，夜有所省，周有温故，月有知新，夏读三伏，冬练三九，日积月累，持之 以恒，武功必有所得，高手终可练成。

小型研讨会 学而时习之，不亦说乎。每季度我们将举办一次小型研讨会(R&D symposium)，同事间或切磋技术，或 分享读书，或交流管理，亦或聆听外部嘉宾的技术报告、感受他们的人生经历，三人行必有吾师焉。我们 不变的宗旨是“Enhance Learning and Improve Effectiveness”。

员工大会
