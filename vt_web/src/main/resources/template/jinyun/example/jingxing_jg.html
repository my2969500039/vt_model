单晶结构解析

结构决定性质，为了确定化合物的结构，20世纪以来红外、紫外、质谱、核磁、单 晶衍射等技术手段纷纷登场。 而在这些技术中，单晶X-ray衍射分析（SC-XRD）为 探究化合物结构提供了一种精准的解决方案。

单晶X射线衍射是确定晶体的原子和分子结构的分析技术，众所周知，晶体结构会导 致入射X射线束向多 个特定方向衍射。通过测量这些衍射光束的角度和强度，可以生 成晶体内电子密度的三维图，进而可 以确定晶体中原子的平均位置以及它们的化学键， 晶体学无序和各种其他信息。

单晶项目方案

  • 培养单晶

    常规的方法主要包括缓慢挥发，气液渗透，缓慢降温与液面扩散

  • 数据收集

    性能优异，高分辨率，高亮度的单晶衍射仪是我们得到高质量的衍射数据的保证

  • 结构解析

    结构解析可以大致分为两步：结构初解与结构精修，最终获得空间全部原子的精确坐标及位移参数

  • 绝对构型验证

    晶云目前同时采用Flack方法与Hooft方法对单晶结构数据中绝对构型判定进行验证，保证绝对构型确证的准确性

MicroED结构解析

微晶电子衍射（MicroED）是一种新颖的3D衍射技术。

Micro ED与X射线衍射（XRD）的不同之处在于它使用电子束从目标样品中获得衍射图样。将样品相对于入射电子束倾斜，可以从多个晶体取向获得衍射图样。 因此，晶体结构可以在三个维度上映射，通过衍射数据缩减和结构确定的步骤，可以在三个维度上绘制晶体结构以及分子结构。

案例解析

1

单晶项目案例

单晶培养 结合化合物的结构特性和晶云丰富的单晶培 养经验，通过优化培养方法，尽可能在有限 的物料和时间条件下得到高质量单晶样品。

结构确证 准确解析化合物结构、手性和结晶水 / 溶剂 信息。

背景信息

  • 某抗生素化合物A有10+个手性中心，晶型盐型现象复杂，希望通过单晶培养和解析来确证绝对构型和晶型。

  • 化合物A分子量大(>1200)，结晶难度大，晶云通过细致专业地设计单晶培养试验条件，多轮次不断优化得到单晶样品，并成功解析。

晶云方案

  • 1. 采用缓慢挥发、汽液渗透、缓慢降温和液面扩散等方法设置单晶培养试验，在高温汽液渗透条件下得到了可用于解析的单晶样品。
  • 2. 收集单晶数据并解析，结果显示该晶体属正交晶系，P212121空间群，晶胞中包含两个化合物A单体，其中一个为內盐，另一 个为盐酸盐，该晶体为化合物半盐酸盐。同时化合物的绝对构型得到确证，Flack参数为0.037(9)，Hooft参数为0.047(8)。

体视显微镜和化合物单晶的显微镜图

Micro ED案例

技术优点 1.适用于小尺寸的样品(< 200 nm)。
2.强大的库仑相互作用可识别轻原子，例如 H和Li。
3.由于较短的波长和较高的衍射强度，空 间分辨率更高。
4.同时反馈衍射和成像信息。

背景信息

  • 某示范化合物多晶型现象复杂，其中晶型X的鉴定存在挑战。

  • 晶型X单晶培养困难，晶体尺寸不足以用于单晶测试和解析。

晶云方案

  • 晶云使用单晶培养试验中得到的粉末样品进行Micro ED测试，快速解析结构，确定晶型X为管道溶剂合物。
