晶型研究

药物晶型是连接药物分子与最终药物产品的一座重要桥梁。同一药物的不同晶型具有 不同的物理化学性质（理化性质），如溶解度、稳定性、引湿性、流动性等，进而会 影响到药品的质量、生产、药效以及安全性…
