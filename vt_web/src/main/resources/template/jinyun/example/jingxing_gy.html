结晶工艺开发流程

结晶是制药生产中必不可少的操作。一方面，大部分活性药物成分（API）是以固 体形式进行开发，其中又以晶体形式最为常见；另一方面，在整个制药行业的历史 中，结晶一直是最重要的分离和纯化手段之一。

结晶工艺开发是一个系统工程，建立在充分理解目标晶型的理化性质以及晶型间相 互转化关系的基础上。选择合适的结晶体系和结晶方式，辅以在线监控设备，能够 有效控制目标晶型的选择性结晶，并确保纯度 ( 包括手性纯度 ) 、颗粒属性、收率、 溶剂残留等各项指标满足要求。

关键工艺参数-成核与生长

控制结晶过程可以从调节晶体成核或生长的角度入手，通常需要同时控制这两种因素。从总体结果的角度出发，必须对每个单元操作进行评估，以确定最关键的单元操作，进而确定成核或生长哪一个对结 晶过程起主导作用。

对于结晶过程的控制，传统观点多关注于成核，因为最初形成的晶核的种类、数量和尺寸至关重要。然而，在生长占主导地位之前有多个因素会影响到晶核产生、成核速率和晶核数量，往往导致成核过程 难以控制因此，基于最终原料药物理属性的控制要求，须将结晶工艺的重点从成核控制转向生长控制，其中最关键的控制因素即为过饱和度和晶种。

案例解析

1

晶型控制案例

通过研究确定晶型转化关系，控制晶种添加 量和时机，优化干燥条件等方式，尽可能保 证目标晶型在结晶制备的全过程中均处于热 力学稳定条件下。对于热力学亚稳晶型，则 通过控制动力学参数，最大程度降低转晶的 风险。

背景信息

  • 国内某Biotech公司选择化合物A的晶型B进行开发，工艺产品为小颗粒不规则样品，样品流动性差，堆密度低。

  • 晶云通过系统地溶剂筛选和结晶工艺优化，找到适合晶型B生长的溶剂体系和工艺条件，通过对样品颗粒的控制，得到流动性好堆密度高的产品。

晶云方案

  • 1.经过筛选一系列工艺溶剂，找出适合晶型B生长的数个候选溶剂体系。
  • 2.收集晶型B在候选溶剂体系的基础数据，包括溶液稳定性、溶解度和亚稳区间，进一步选出优选体系。
  • 3.在成功摸索出晶型B的稳定工艺操作区间后，得到了棒状晶型B样品，堆密度提升3倍。

A:通过溶解度确定不同晶型在特定溶剂比例下的稳定性关系

B:最终产品PLM图

2

纯度提升案例

利用化合物与杂质的溶解度等差异，通过优 化结晶条件，如晶种、温度、溶剂体系等， 达到除杂提纯的效果，高效、可控地满足质 量要求。

背景信息

  • 国内某制药企业在开发某抗生素类化合物的过程中，发现样品在长期稳定性和加速稳定性测试过程中会发生明显的颜色变化不符 合药典对该品种溶液颜色的要求。

  • 晶云通过结晶工艺的系统性研究、开发以及优化，在解决化合物变色问题的基础上进一步提高了化合物的性质（包括结晶度、稳 定性等），更好地满足药品质量要求，供后续申报上市的使用。

晶云方案

  • 1.通过不同结晶度样品的制备和固态稳定性评估等一系列研究发现，该化合物在稳定性测试中的颜色变化程度与其结晶度的高低成负 相关。起始样品的结晶度越低，其稳定性样品的颜色变化越明显，推测是其含有的微量有色杂质所导致。
  • 2.经过系统性的结晶工艺开发和优化（包括但不限于溶剂筛选、晶种优化、过饱和度控制等），晶云为客户提供了一套可以稳定得到 结晶度优于市售和客户自制产品的工艺。由该工艺所生产的产品，其长期和加速稳定性得到极大的改善，溶液颜色符合药典的要求， 满足了质量控制和后续申报要求。

三元相图指导手性拆分案例

通过与手性试剂的结合，将对应异构体转变 为非对映异构体，进一步探索非对映异构体 在不同溶剂或溶剂体系中的溶解度 / 析晶速 率差异，从而优化得到高效的拆分方案。

背景信息

  • 国外某综合型公司，在开发目标化合物的过程中需要拆分一对非对映异构体D/U（D为目标产物）。前期已尝试过化学拆分等手 段，但结果并不理想。希望晶云能为该化合物开发出一套拆分体系和相应的拆分方法，保障最终产品的ee值在98%以上。

  • 晶云通过测定三元相图，准确选择了合适的溶剂体系和温度，并在工艺验证实验中成功得到ee值符合客户要求的非对映异构体D， 理论推算与实际结果相互映证。

晶云方案

  • 1.研究的重点在于寻找合适的拆分体系。根据化合物的性质，发现将D和U在特定的拆分体系中可以得到其消旋体（命名为X）。
  • 2.在开发出拆分体系后，进一步收集非对映异构体D和消旋体X不同比例混合物在特定温度下的溶解度数据，绘制出三元相图。最终 根据相图，框定了稳定的拆分区间，确定了在该条件下的eutectic ee为98.2%，符合客户的期待。
  • 3.根据三元相图的数据以及初步摸索出的条件进行拆分工艺的验证。在百克规模下，得到ee值为98.1%的非对映异构体D样品，为客 户的后续生产提供了技术支持。

4
