晶型研究 晶型研究贯穿于药物发现和开发的整个过程。晶云药物是晶型研究领域 的领导者，为全球制药企业提供定制化的晶型研究方案，推动药物早日 开发上市。

晶型研究贯穿药物研发全流程

从药物发现阶段的先导化合物筛选及优化，临床前研究，临床试验，直至最终药物上市，都离不开晶型研究的支持。 药物晶型是连接药物分子与最终药物产品的一座重要桥梁。同一药物的不同晶型具有不同的理化性质，如溶解度、稳定性、引 湿性、流动性等，进而会影响到药品的质量、生产、药效以及安全性。

结合实际需求提供定制研究方案

晶云药物根据化合物进展的阶段和实际研究过程中遇到 的挑战定制合理的技术方案，迄今为止已为超过500家 客户和超过1000个新药化合物提供了药物晶型研发的 专业技术解决方案。

如果化合物理化性质(如溶解度、稳定性等)能够满足开 发需求，优先考虑采用游离态进行多晶型筛选，并选出 理化性质优良、工艺可放大的优势晶型。

如果化合物性质理化需要改善，盐型和共晶的筛选被纳 入考虑，选择能够提升溶解度、稳定性等性质的盐或共 晶再进行多晶型筛选和评估，最终选出优势晶型供客户 进一步开发。

案例分析

1

多晶型筛选案例

筛选潜在晶型 晶云药物依托晶型筛选领域的经验积累，根 据化合物特性和客户需求优化筛选方案，将 每一个筛选试验的价值最大化，发现尽可能 多的化合物晶型。

确定“优势晶型” 在药物开发过程中，为了寻找到适于开发的 “优势晶型”，通常会选择热力学上稳定的 晶型用于开发，以尽可能降低开发过程中的 晶型转变风险。

背景信息

  • 某国内新药开发公司希望为处于临床前阶段的化合物， 开展全面的多晶型筛选工作，以期望选择理化性质较优的晶型用于后续开发。

  • 前期客户开展了在少数溶剂体系中的初步筛选，发现了化合物 有可能存在水合物以及溶剂合物。

晶云方案

  • 初始样品表征：对化合物进行基础表 征，包括固态表征、 粗略溶解度等，用于 指导试验设计。

  • 多晶型筛选实验： 采用多种固相转变和 溶液结晶方法设置多 晶型筛选试验，以得 到尽可能多的晶型。

  • 多晶型固态表征和鉴定： 综合各种固相研究方 法，对筛选试验得到 晶型进行鉴定，明确 可开发的潜在晶型。

  • 晶型间相互转化关系研究： 晶型间相互转化关系 研究，确定各晶型间 的热力学稳定性关系 （含关键水活度）。

  • 综合评估稳定性、溶解度、引湿性等： 综合考察优势晶型的 物理化学性质，进而 评估该优势晶型在后 续开发中的风险。

A:筛选试验设计（不同颜色代表不同溶剂）

B:筛选结果对比

晶云筛选策略(根据化合物特性设计合适的结晶溶剂和方法)提高找到潜在晶型可能性

结论

  • 1. 根据化合物特性设计的全面多晶型筛选试验共得到了20个新晶型。
  • 2. 通过理化性质表征和晶型鉴定发现的新晶型中包含6个无水晶型、3个水合物11个溶剂合物。
  • 3. 晶型转化关系研究结果显示无水晶型Ⅰ室温下特定的水活度条件中稳定，且具有良好物理和化学稳定性、几乎无引湿性、符合 开发需求的溶解度，推荐为优势晶型用于后续开发和研究。

2

盐型 / 共晶筛选案例

盐型 / 共晶筛选案例 针对越来越频繁出现的难溶性药物，利用盐 或共晶的形式来提高其溶解度，从而提高药 物的生物利用度及成药性，是当前的一个重 要开发策略。

优选盐型/共晶 对候选盐型 / 共晶进行全方位评估，包括物 理和化学稳定性、溶解度、引湿性等，推荐 综合属性最合适的盐型 / 共晶用于进一步开 发。

背景信息

  • 某国外Biotech公司希望为其在临床前阶段的化合物开展全面的盐型/共晶筛选。

  • 化合物为弱碱性(pKa ~2.3)，前期曾初步开展了游离碱的多晶型筛选工作，发现了若干晶型(>10)，后选择了游离碱晶型I作为 开发晶型。

  • 由于前期开发的游离碱晶型生物利用度相对受限，该客户希望开展完整的盐型/共晶筛选工作，以评估是否有理化性质（尤其是 溶解度更优）的晶型，同时也对盐型/共晶进行全面的保护。

晶云方案

  • 对化合物进行基础表 征，包括固态表征、 粗略溶解度、pKa等， 用于指导试验设计。

  • 采用多种不同方法、 配体和溶剂体系实施 若干数量的盐型 / 共 晶筛选试验。

  • 综合评估各个盐型的 理化性质和可开发性， 对性质较优的盐型或 共晶进行放大制备。

  • 通过单晶结构解析、 XPS (X射线光电子能 谱）等分析测试手段 鉴别盐和共晶。

  • 进一步评估候选盐 / 共晶的性质和潜在的 风险，推荐最适合开 发的盐型。

结论

  • 1. 综合盐型筛选和表征的结果，酸的安全级别，多晶型现象等因素，选出候选盐型。
  • 2. 进一步评估候选盐型的引湿性，固态稳定性和溶解度并与游离碱晶型I进行综合评价，选出各方面性质适合的优势盐型(见下图)。
  • 3. 通过单晶培养与解析，确证盐型C为盐而非共晶。
  • 4. 为客户申请专利保护提供数据支持，延长化合物生命周期
